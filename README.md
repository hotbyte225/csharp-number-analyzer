# Number Analyzer (Console App)

A simple C# console application that analyzes numbers entered by the user.

## Features
- Counts even and odd numbers
- Counts positive, negative, and zero values
- Finds the maximum and minimum values
- Calculates the average value

## Tech Stack
- C#
- .NET Console Application

## Getting Started
1. Clone the repository
2. Open the project in Visual Studio or VS Code
3. Run the application

## Example
Input:
5  
1  -3  4  0  7

Output:
Even numbers: 2  
Odd numbers: 3  
Positive numbers: 2  
Negative numbers: 1  
Zero values: 1  
Max value: 7  
Min value: -3  
Average value: 1.80
